## About

Cleopatra Delights is a Jacksonville-based global bakery offering handcrafted desserts from cultures around the world. Every item is made with care — from Dubai Chocolate Cookies to Banana Pudding Flans, Biscoff Cheesecakes to Ferrero Rocher Cake Bars.

---

## Menu Highlights

- **NYC Cookies** — 14 flavors including Dubai Chocolate, Red Velvet S'more, Kinder Bueno
- **Cookie Cakes** — 6 flavors
- **Brownies** — 6 flavors including Rocky Road and Red Velvet Cream Cheese
- **Blondies, Cheesecakes, Shortcakes, Cake Bars, Flans, Cakes** — and more

## Tech Stack

```
Framework       Next.js 15 (App Router)
Language        TypeScript
Styling         Tailwind CSS
Animation       Framer Motion
Icons           Lucide React
Deployment      Vercel
```

## Running Locally

```bash
git clone https://github.com/loki128/cleopatra-delights.git
cd cleopatra-delights
npm install
npm run dev
# Open http://localhost:3000
```
